# Contributing

## Setup

- Install Node.js (>= 18) and enable Corepack.
- Install deps:

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

## Tests

```bash
pnpm run typecheck
pnpm test
```

## Security

Never commit secrets or `figd_*` tokens. Use `.figma-mcp.local.json` or environment variables instead.
